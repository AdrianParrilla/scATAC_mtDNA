# scATAC_mtDNA
Implementation of single-cell ATAC-seq protocol modified to infere mitochondrial DNA heteroplasmy levels and to enable cellular lineage tracing from human PBMCs.  
The idea and analisys pipeline are taken from the publication: Lareau, C.A., Liu, V., Muus, C. et al. *Mitochondrial single-cell ATAC-seq for high-throughput multi-omic detection of mitochondrial genotypes and chromatin accessibility*. Nat Protoc 18, 1416â€“1440 (2023). https://doi.org/10.1038/s41596-022-00795-3

- mgatk: mitochondrial genome analysis toolkit (https://github.com/caleblareau/mgatk)
